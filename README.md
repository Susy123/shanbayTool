shanbayTool
--------------------------------
##功能：
1、去除 theguardian 次要信息，保留主题的标题、图片和正文；
2、根据屏幕自动分页，无需滑动鼠标或者拖动滚动条；
3、通过扇贝的 API 查词释义和发音；
4、响应式布局；通过chrome resize window进行测试；

##使用方法：
安装扩展，刷新要查词的页面，功能即可生效；


##代码修改：
依赖Node.js

cd shanbayTool
npm install   
gulp sass

##项目部署：
进入Chrome的扩展程序设置页面，点击“加载正在开发的扩展程序”，选择shanbayTool目录

修改代码之后,直接在浏览器的扩展程序页面，点击“重新加载”即可
